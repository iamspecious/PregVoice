PregVoice is an open-source project that aims to make pregnancy tests accessible for visually impaired individuals. The app will use a smartphone’s camera to detect test results and announce them aloud, offering a simple and privacy-focused solution.

Wireframe

This is a written description of the app’s wireframe:

  • Open the app.
  • Prompt to line up the test within a large box.
  • Take the photo.
  • Display a 'processing' screen with a voice command.
  • Articulate the result.
  • Prompt whether the user wants to retry with another test or rehear the response.
